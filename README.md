# Family World Cup 2026 GitHub Shared View

Upload `index.html` and `data.json` to GitHub Pages.

Public view link: normal Pages URL.
Dietrich edit link: add `?edit=1` to the end.

After editing visually, click `Export data.json`, then replace `data.json` in GitHub.


Correct predictions are highlighted in red automatically after a valid final score is entered, for example 2-0 or 1-1.
